# The Impact of ESG Performance on Long-Term Corporate Value

## 研究背景

ESG（环境、社会、治理）投资近年来受到广泛关注，但ESG表现究竟如何影响企业长期价值，
学术界尚无定论。本研究通过构建多年期面板数据，实证检验ESG绩效与企业价值之间的动态关系。

## 数据

- 数据来源：Bloomberg Terminal & Refinitiv Eikon
- 样本规模：780+ 家上市公司
- 时间跨度：5年面板数据
- 企业价值代理变量：Tobin's Q
- 核心自变量：ESG综合评分及E、S、G三个子维度评分

## 方法论

**模型选择**
通过 Hausman 检验判断固定效应与随机效应模型的适用性，
最终选择随机效应模型（Random Effects），并使用稳健标准误控制异方差问题。

**核心模型**
- 基准回归：ESG对Tobin's Q的影响
- 动态效应：加入ESG × Year交互项，捕捉ESG影响的时间路径
- 中介效应：检验R&D投入、ROA作为中介变量的传导机制
- 子维度分析：分别检验E、S、G对企业价值的独立影响
- 稳健性检验：三组不同控制变量组合验证结果稳定性

**数据处理流程**
1. 宽格式数据转长格式，再重构为标准面板结构
2. Pearson相关性检验 + 热力图，识别多重共线性风险
3. 行业虚拟变量控制行业固定效应
4. 年份交互项捕捉动态时间效应

## 主要发现

ESG绩效与企业长期价值呈 **U型非线性关系**：
- 短期内（第1-2年）ESG投入对Tobin's Q系数为**负**
- 从第3年起系数转为**显著正向**，且逐年增强

这一发现说明ESG投资存在明显的滞后效应，
对投资者而言，短期内不应以ESG系数为负而否定其长期价值。
